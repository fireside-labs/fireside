# Acknowledgments

Fireside is built on the shoulders of incredible open-source projects.

## Conceptual Inspiration

- **[OpenClaw](https://github.com/openclaw)** — The multi-node AI agent concept that inspired Fireside's mesh architecture. We started here, then rebuilt from scratch with a focus on consumer UX, overnight learning, and one-click setup.

## Inference Runtimes

- **[oMLX](https://github.com/jundot/omlx)** — LLM inference server with continuous batching & SSD caching for Apple Silicon. Apache 2.0 License.
- **[llama.cpp](https://github.com/ggml-org/llama.cpp)** — LLM inference in C/C++ for NVIDIA and CPU. MIT License.

## AI Foundations

- **[MLX](https://github.com/ml-explore/mlx)** by Apple — Machine learning framework for Apple Silicon.
- **[mlx-lm](https://github.com/ml-explore/mlx-lm)** by Apple — Language model inference for MLX.

## Web Browsing

- **[PinchTab](https://github.com/pinchtab/pinchtab)** — High-performance browser automation using accessibility trees. Our `browse` plugin's tree-based parser was inspired by PinchTab's approach of extracting structured content from the accessibility tree instead of raw DOM.

## Built With

- [FastAPI](https://github.com/tiangolo/fastapi) — Backend API framework
- [Next.js](https://nextjs.org) — Dashboard framework
- [Tailwind CSS](https://tailwindcss.com) — Dashboard styling
- [Tauri](https://tauri.app) — Desktop app framework
- [BeautifulSoup4](https://www.crummy.com/software/BeautifulSoup/) — HTML parsing
